🎯 Estado del pipeline de datos

El dashboard ya usa las fuentes correctas: Reservas_Asignadas (3.592 reservas, 2018–2026) y Facturas_Detallado (20.954 líneas, 2016–2026). Para mantenerlo actualizado solo necesitas descargar estos dos CSVs de Redforts periódicamente y procesarlos con el script de abajo.

Instrucciones paso a paso
Configuración única · ~20 minutos

  1. Exporta desde Redforts

    En Redforts: Reservas → Exportar CSV y Facturas → Exportar detallado. Selecciona el rango de fechas que quieras (recomendado: siempre exportar todo el histórico).

  2. Abre Apps Script en tu Sheet

    En tu Google Sheet: Extensiones → Apps Script. Borra el contenido y pega el código de abajo.

  3. Sube los CSV a Google Drive

    Crea una carpeta llamada Redforts_Data en tu Drive. Sube los dos archivos descargados de Redforts.

  4. Ejecuta el script

    Pulsa ▶ Run o usa el nuevo menú 🏨 La Fronda que aparecerá en tu Sheet. El script crea automáticamente las hojas RESERVAS y FACTURAS.

  5. Refresca cuando quieras

    Cada vez que descargues datos nuevos de Redforts, súbelos a la misma carpeta y ejecuta el script. Solo añade filas nuevas, nunca duplica.

Script Google Apps Script — Importador Redforts → Google Sheets
Pega en Extensiones → Apps Script · funciona con los dos CSVs de Redforts

// ============================================================ // LA FRONDA — Importador Redforts CSV → Google Sheets // ============================================================ const FOLDER_NAME = 'Redforts_Data'; // Nombre de tu carpeta en Drive const SHEET_RES = 'RESERVAS'; const SHEET_FAC = 'FACTURAS'; function onOpen() { SpreadsheetApp.getUi() .createMenu('🏨 La Fronda') .addItem('📥 Importar datos de Redforts', 'importarTodo') .addItem('📊 Ver resumen de datos', 'resumenDatos') .addToUi(); } function importarTodo() { const folders = DriveApp.getFoldersByName(FOLDER_NAME); if (!folders.hasNext()) { SpreadsheetApp.getUi().alert('❌ No encontré la carpeta "' + FOLDER_NAME + '" en tu Drive.\nCréala y sube los CSV de Redforts.'); return; } const folder = folders.next(); const files = folder.getFiles(); let importados = 0; while (files.hasNext()) { const file = files.next(); const name = file.getName().toLowerCase(); if (name.includes('reservas') || name.includes('reserva')) { importarCSV(file, SHEET_RES); importados++; } else if (name.includes('factura') || name.includes('facturas')) { importarCSV(file, SHEET_FAC); importados++; } } if (importados === 0) { SpreadsheetApp.getUi().alert('⚠️ No encontré archivos de reservas o facturas en la carpeta.\nAsegúrate que los nombres incluyan "reservas" o "facturas".'); } else { SpreadsheetApp.getUi().alert('✅ ' + importados + ' archivo(s) importado(s) correctamente.'); } } function importarCSV(file, sheetName) { const ss = SpreadsheetApp.getActiveSpreadsheet(); let sheet = ss.getSheetByName(sheetName); if (!sheet) sheet = ss.insertSheet(sheetName); const content = file.getBlob().getDataAsString('UTF-8'); const rows = Utilities.parseCsv(content, ';'); if (rows.length < 2) return; // Construir set de claves existentes para evitar duplicados const existing = sheet.getLastRow() > 0 ? sheet.getDataRange().getValues() : []; const existingKeys = new Set(existing.slice(1).map(r => r[0] + '|' + r[1])); // Si la hoja está vacía, añadir cabecera if (sheet.getLastRow() === 0) { sheet.appendRow(rows[0]); } let added = 0; for (let i = 1; i < rows.length; i++) { const key = rows[i][0] + '|' + rows[i][1]; if (!existingKeys.has(key)) { sheet.appendRow(rows[i]); existingKeys.add(key); added++; } } Logger.log(sheetName + ': ' + added + ' filas nuevas añadidas.'); } function resumenDatos() { const ss = SpreadsheetApp.getActiveSpreadsheet(); const sheetR = ss.getSheetByName(SHEET_RES); const sheetF = ss.getSheetByName(SHEET_FAC); const nRes = sheetR ? sheetR.getLastRow() - 1 : 0; const nFac = sheetF ? sheetF.getLastRow() - 1 : 0; SpreadsheetApp.getUi().alert( '📊 Resumen de datos en tu Sheet:\n\n' + '• Reservas: ' + nRes + ' registros\n' + '• Líneas de factura: ' + nFac + ' registros\n\n' + 'Para actualizar, descarga los CSV de Redforts,\nsúbelos a la carpeta "' + FOLDER_NAME + '" y ejecuta de nuevo.' ); }
